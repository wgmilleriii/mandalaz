Mandala library

An extension of drawlib.js

Let's work smarter, not harder.
Mandalas are constructed on an SVG object. There can be multiuple SVG objects per page (like this one). To get started, assign a variable to reference the object on which to draw. Here, we'll begin drawing on the SVG object with the identifier id="svg0" and draw a grid. 
    
    s=document.getElementById("svg0");
    s.grid(100,20);
Next, we'l create a group with g=s.makeGroup(), and add two circles to the group. Mandala-ing is as simple as calling g.mandala(6); 
    
    s=document.getElementById("svg0");
    s.grid(100,20);
    
    g=s.makeGroup();
    g.circle(0,100,5,"thin_grey");
    g.circle(0,100,20,"thin_grey");
    g.mandala(6);
The g.mandala(6) code takes two parameters: the number of points, and an offset. Here's a new group with a path, and an offset of .5: 
    
    g=s.makeGroup();
    g.circle(0,100,5,"thin_grey");
    g.circle(0,100,20,"thin_grey");
    g.mandala(6);

    g=s.makeGroup();
    g.circle(0,100,10,"thin_grey");
    g.mandala(6, .5);

    g=s.makeGroup();
    p=g.path();
    p.setPath("M 100 100 Q 120 120 140 100");
    g.mandala(16);
